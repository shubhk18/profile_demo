# 🚀 C++ System Showcase

A **C++ backend-driven terminal experience** that simulates a
high-performance system with real concurrency and live metrics.

------------------------------------------------------------------------

## ✨ What it does

-   🧵 Runs a **multi-threaded workload**
-   ⚡ Measures **latency (µs)** and **throughput (ops/sec)**
-   🖥️ Streams output as a **Linux-style terminal UI**
-   🌐 Served directly from a **C++ HTTP server**

------------------------------------------------------------------------

## 🧠 Why it's different

-   No frontend frameworks\
-   No Node.js\
-   Pure **C++ + Linux-style systems thinking**

👉 Designed to *show*, not just tell, engineering capability.

------------------------------------------------------------------------

## 🛠️ Run locally

``` bash
g++ server.cpp -std=c++17 -O2 -pthread -o server
./server
```

Open: http://localhost:8080
